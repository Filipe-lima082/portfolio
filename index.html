Conhecimentos

Git Version Control

Git é um sistema de controle de versão de arquivos. Através deles podemos desenvolver projetos na qual diversas pessoas podem contribuir simultaneamente.

JavaScript

JavaScript é uma linguagem de programação que permite a você implementar itens complexos em páginas web.

React Js

React JS é uma biblioteca JavaScript para a criação de interfaces de usuário — ou UI (user interface).

HTML 5

HTML é uma das linguagens que utilizamos para desenvolver websites. O acrônimo HTML vem do inglês e significa Hypertext Markup Language.

CSS 3

CSS é chamado de linguagem Cascading Style Sheet e é usado para estilizar elementos escritos em uma linguagem de marcação como HTML.

Sass

Sass é uma linguagem de extensão do CSS. Com ele, podemos criar variáveis, separar o nosso CSS em módulos, criar funções, implementar lógica de programação, e muito mais.
